### Partial Derivatives

Just like derivatives measure change in single variable functions, partial derivatives measure the change in one direction while holding the others constant. 

For $f(x,y)$:
$$f_x = \frac{\partial f}{\partial x}\quad (Treat \ y \ as \ constant),$$
$$f_y = \frac{\partial f}{\partial y}\quad (Treat \ x \ as \ constant).$$

This lets you analyze function change per input.

#### Example

$$f(x,y)=3x^2y+4y^3$$
Find $f_x$ treating y as a constant.
$$f_x=6xy$$
This is because $4y^3$ is consant so it fades away. The y in $3x^2y$ cannot fade since it is manipulating x. 

Doing the same for $f_y$ treating x as a constant.
$$f_y=3x^2+12y^2$$
