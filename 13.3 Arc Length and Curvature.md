### Arc Length
A curve defined by a vector function 
$$r(t) = <x(t), y(t), z(t)>$$
To compute the distance traveled along the curve between t=a and t=b (**arc length**) with:
$$L=∫_a^b​|r^′(t)|dt$$
where 
$$​|r^′(t)| = \sqrt{[x^′(t)]^2 + [y^′(t)]^2 + [z^′(t)]^2}$$
#### Example: Find the arc length
$$r(t)=⟨3t,  4t,  0⟩$$$$0≤t≤2.$$
Step 1: Find the derivative
$$r^′(t)=⟨3,4,0⟩$$
Step 2: Find its magnitude (speed)
$$|r^′(t)|=\sqrt{3^2+4^2}=5$$

Step 3: Integrate speed
$$L=∫_0^25 dt$$$$L=5(2)=10$$
The arc length is 10 units.
### Important Unit Vectors
#### Unit Tangent Vector
$$T(t)=\frac{r^′(t)}{|r^′(t)|}$$

#### Unit Normal Vector
$$N(t) = \frac{T^′}{|T^′|}$$
##### Example
$$r^′(t) = [1,1,t]$$
$$T(t) = \frac{[1,1,t]}{\sqrt{2+t^2}}$$
$$T(t) = (2+t^2)^{-1/2}[1,1,t]$$
Using the product rule we can find $T^′(t)$
Let:
$$u = (2+t^2)^{-1/2}$$
$$w=[1,1,t]$$
Then:
$$T^′=u^′w+uw^′$$
Compute derivatives:
$$u^′=-\frac{1}{2}(2+t^2)^{-3/2}(2t)=-\frac{t}{(2+t^2)^{3/2}}$$
$$w^′=[0,0,1]$$
Then recombine and simplify
$$T^′=--\frac{t}{(2+t^2)^{3/2}}[1,1,t] + \frac{1}{\sqrt{2+t^2}}[0,0,1]$$
$$=\frac{[-t,-t,-t^2]+[0,0,2+t^2]}{(2+t^2)^{3/2}}$$$$T^′(t)=\frac{[-t,-t,2]}{(2+t^2)^{3/2}}$$
Find $|T^′|$:
$$|T^′|=\frac{\sqrt{2}}{2+t^2}$$
Now find unit normal vector
$$N(t) = \frac{T^′}{|T^′|}$$
$$=\frac{[-t,-t,2]}{(2+t^2)^{3/2}} * \frac{2+t^2}{\sqrt2}$$
$$=\frac{[-t,-t,2]}{\sqrt2 \sqrt{2+t^2}}$$
### Curvature
Curvature is how sharply a curve bends at a defined point
- Straight line curve is 0
- For a space curve one formula for curvature is:

$$\kappa = \frac{|\mathbf{r}'(t) \times \mathbf{r}''(t)|}{|\mathbf{r}'(t)|^3}$$
#### Curvature Example
Solve for general point =(0,0,0)
Let $$r(t)=⟨t ,t^2, t^3⟩$$
Step 1. First Derivative
$$r^′(t)=⟨1,2t,3t^2⟩$$
$$|r^′(t)| = \sqrt{1^2 + (2t)^2 + (3t^2)^2} = \sqrt{1+4t^2 + 9t^4}$$
Step 2. Second Derivative
$$r^{′′}(t)=⟨0,2,6t⟩$$
Step 3. Cross Products
$$r^′(t) \times r^{′′}(t) = ⟨6t^2,-6t,2⟩$$
$$|r^′(t)\times r^{′′}(t)| =\sqrt{(6t^2)^2 + (-6t)^2 + (2)^2} =\sqrt{36t^4 + 36t^2+4}$$
Step 4. Combine
$$\kappa=\frac{|\mathbf{r}'(t) \times \mathbf{r}''(t)|}{|\mathbf{r}'(t)|^3}=\frac{\sqrt{36t^4 + 36t^2+4}}{(1+4t^2+9t^4)^{3/2}}$$
Step 5. Solve at point
$$\kappa(0) =2$$
